This is a document including some computational projects on an open-source platform.

\(\textbf {1.}\) Briefly it starts with explaining the basic concepts of some hyperelastic material models and compares the analytical solutions with the numerical results based on finite element method (FEM).

\(\textbf {2.}\) In the second chapter, it goes over how to numerically solve a Multi-Physics problem using FEM.

\(\textbf {3.}\) In the third chapter, details of a Differential Evolution optimization algorithm will be described and later it will be explained how it could be used to implement Inverse FEM for tunning parameters in a numerical model to fit into the experimental data.

\(\textbf {4.}\) The last chapter goes over the theory and implementation of a supervised machine learning technique (Linear Discriminant Analysis) and going over an application in disease diagnosis.

Note

The reader is assumed to be familiar with the basic concepts of continuum mechanics, as well as FEM and coding in Python.
